Formulas para calcular el radio de un circulo

Ahora aprenderemos a calcular el radio de un circulo:




Para calcular el radio de un circulo podriamos usar cualquiera
de estas formulas:

  • Calcular el radio apartir del diametro
  • Calcular el radio apartir del perimetro(Longitud o Circunferencia)
  • Calcular el radio apartir del area

Calcular el radio apartir del diametro

Para calcular el Radio de un circulo apartir del diametro usaremos esta formula:


    Donde
  • R = Radio.
  • D = Diametro.
    • El radio es la mitad del diametro.

Ejemplo:

Imagina que tienes un circulo cuyo diametro es de 30 cm y quieres obtener su radio apartir del diametro. entonces nuestra formula quedara asi:

r = D / 2

A D le daremos el valor de 30cm por lo que nos quedara asi:
r = 30cm / 2 = 15cm

Donde 15cm sera el valor de nuestro radio

Calcular el radio apartir del perimetro(Longitud o Circunferencia)

Para calcular el radio de un circulo usando su perimetro usaremos esta formula:

    Donde
  • R = Radio.
  • P = Perimetro.
  • π(Se llama pi) = 3.1416

Ejemplo:

Imagina que tienes un circulo cuyo perimetro es de 40 cm y quieres obtener su radio apartir del perimetro. entonces nuestra formula quedara asi:

r = P / 2π

Donde P tiene un valor de 40cm y π(pi) vale 3.1416por lo que la formula nos quedara asi:
r = 40cm / 2 x 3.1416 = 6.3661cm

  • Primero mulltiplicamos 2 x π(pi) el cual tiene un valor de 3.1416 y nos dara como resultado 6.2832cm
  • Despuesdividiremos 40 / 6.2832cm y nos dara como resultado 6.3661cm
Donde 6.3661cm sera el valor de nuestro radio

Calcular el radio apartir del area

Para calcular el radio apartir del area usaemos esta formula:

    Donde
  • R = Radio.
  • A = Area.
  • π(Se llama pi) = 3.1416
  • √ = Raíz cuadrada

Ejemplo

Imagina que tenemos un circulo con un Area de 60cm y queremos calcular su radio partiendo de esta area entoncesnuestra formula quedara asi:
r = √(A / π)
Donde A tiene un valor de 60cm y π(pi) vale 3.1416.
r = √(60cm / 3.1416) = 4.3701cm

  • Primero dividiremos los 60cm entre 3.1416 y nos dara como resultado 19.0985cm
  • Despues sacaremos la raiz cuadrada de 19.0985 y nos dara como resltado 4.3701cm
